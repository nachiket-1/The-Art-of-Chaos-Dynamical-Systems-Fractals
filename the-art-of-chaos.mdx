## § 01 · The Butterfly Effect

### 🦋 The Lorenz Attractor

In **1963**, meteorologist Edward Lorenz was running weather simulations when he discovered something that would change science forever. He re-ran a simulation with what he thought was the same starting point — but had rounded `0.506127` to `0.506`. The results were **completely different.**

The three equations below define the Lorenz system — they look innocent:

$$\frac{dx}{dt} = \sigma(y - x)$$

$$\frac{dy}{dt} = x(\rho - z) - y$$

$$\frac{dz}{dt} = xy - \beta z$$

With **σ=10, ρ=28, β=8/3**, these equations never repeat, never settle — they trace an eternal butterfly through phase space.

> ⚡ **KEY INSIGHT**
>
> The Lorenz system is **deterministic** — the same initial conditions always produce the same trajectory.
> Yet it is **unpredictable**, because we can never measure initial conditions with infinite precision.
> This is not a measurement problem. It is a **fundamental feature of the universe.**

```python
# ══════════════════════════════════════════════
#  LORENZ ATTRACTOR — High-res 3D Phase Portrait
# ══════════════════════════════════════════════

def lorenz(t, state, sigma=10.0, rho=28.0, beta=8/3):
    x, y, z = state
    return [sigma*(y - x), x*(rho - z) - y, x*y - beta*z]

t_span = (0, 80)
```

> Classical geometry: 1D lines, 2D surfaces, 3D volumes.

## § 03 · The Road to Chaos

### 📊 The Bifurcation Diagram

Consider the humble **logistic map** — originally a model for animal population dynamics:

$$x_{n+1} = r \cdot x_n \cdot (1 - x_n)$$

- For **r < 3**: population stabilizes to one value
- For **r ≈ 3.45**: oscillates between **2** values
- For **r ≈ 3.54**: oscillates between **4** values
- For **r ≈ 3.57**: **chaos** — infinite, never-repeating values

The period keeps doubling — **1 → 2 → 4 → 8 → 16 → ∞** — until chaos erupts.

> 🔢 **FEIGENBAUM'S CONSTANT**
>
> The ratio between successive bifurcation intervals always converges to **δ ≈ 4.669201...**
> This constant appears in **every** smooth one-dimensional chaotic map — universally.
> It is as fundamental to chaos as **π** is to circles or **e** is to growth.

```python
# ══════════════════════════════════════════════
#  BIFURCATION DIAGRAM — Pixel-perfect rendering
# ══════════════════════════════════════════════

def bifurcation_data(r_min=2.5, r_max=4.0, r_res=4000, iterations=1200, skip=800):
    r_vals = np.linspace(r_min, r_max, r_res)
    x = 0.5 * np.ones(r_res)
    r_points, x_points = [], []
    for i in range(iterations):
        x = r_vals * x * (1 - x)
```

## § 04 · The Most Complex Object in Mathematics

### ∞ The Mandelbrot Set

$$z_{n+1} = z_n^2 + c$$

That's it. **One line.** Applied repeatedly to complex numbers `c`, this map either stays bounded (inside the set) or escapes to infinity. The boundary between these two fates is **infinitely complex** — the same level of detail at every zoom level, forever.

The Mandelbrot set has:
- **Hausdorff dimension = 2** — its boundary has infinite length
- **Area ≈ 1.506484** — finite, yet surrounded by infinite complexity
- **Self-similarity** — zoom anywhere on the boundary and find echoes of the whole

> ∞ **INFINITE COMPLEXITY**
>
> You could zoom into the Mandelbrot set for a **million years** and still find new structures —
> spirals containing spirals, each surrounded by miniature copies of the entire set.
> This emerges from a single rule applied over and over.

```python
# ══════════════════════════════════════════════
#  MANDELBROT SET — Smooth iteration coloring
# ══════════════════════════════════════════════

def mandelbrot(c, max_iter=256):
    z = np.zeros_like(c, dtype=complex)
    iters = np.zeros(c.shape, dtype=float)
    mask = np.ones(c.shape, dtype=bool)
    for i in range(max_iter):
        z[mask] = z[mask]**2 + c[mask]
        escaped = mask & (np.abs(z) > 2.0)
        if np.any(escaped):
            iters[escaped] = i + 1 - np.log2(np.log2(np.abs(z[escaped])))
        mask &= ~escaped
    return iters

def render_region(x_min, x_max, y_min, y_max, width=1200, height=900, max_iter=512):
    x = np.linspace(x_min, x_max, width)
    y = np.linspace(y_min, y_max, height)
    C = x[np.newaxis, :] + 1j * y[:, np.newaxis]
    return mandelbrot(C, max_iter)

fig, axes = plt.subplots(1, 3, figsize=(18, 7))
fig.patch.set_facecolor(PALETTE['bg'])

regions = [
    (-2.5, 1.0, -1.25, 1.25, 512, 'Full Mandelbrot Set'),

    (-0.5555, -0.5505, 0.6265, 0.6305, 1024, 'Zoom Level 2: Infinite Spirals'),
]

for ax, (xmin, xmax, ymin, ymax, mi, title) in zip(axes, regions):
    print(f"⟳  Rendering: {title}...", end=" ", flush=True)
    img = render_region(xmin, xmax, ymin, ymax, 900, 700, mi)
    display_img = np.sqrt(np.sqrt(img))
```

## § 05 · Julia Sets — A Family of Fractals

### 🔮 Julia Set Gallery

The Mandelbrot set is essentially a **map of Julia sets**. For every point `c` in the complex plane, there is a corresponding Julia set — the fractal boundary for that constant.

- If `c` is **inside** the Mandelbrot set → Julia set is **connected** — one intricate piece
- If `c` is **outside** → Julia set **shatters** into a Cantor dust of disconnected points

The Mandelbrot set is literally a **catalogue of all possible Julia sets.**

Each of the 8 fractals below comes from a different value of `c` — same formula, completely different universe:

```python
# ══════════════════════════════════════════════
#  JULIA SET GALLERY — 8 Parametric Fractals
# ══════════════════════════════════════════════

def julia(c_const, x_min=-1.6, x_max=1.6, y_min=-1.2, y_max=1.2,
          width=700, height=550, max_iter=300):
    x = np.linspace(x_min, x_max, width)
    y = np.linspace(y_min, y_max, height)
    Z = x[np.newaxis, :] + 1j * y[:, np.newaxis]
    C = np.full_like(Z, c_const)
    iters = np.zeros(Z.shape, dtype=float)
    mask = np.ones(Z.shape, dtype=bool)
    for i in range(max_iter):
        Z[mask] = Z[mask]**2 + C[mask]
        escaped = mask & (np.abs(Z) > 2.0)
        if np.any(escaped):
            iters[escaped] = i + 1 - np.log2(np.maximum(np.log2(np.abs(Z[escaped])), 1e-10))
        mask &= ~escaped
    return iters
```

## § 06 · Measuring Chaos

### 🌡️ Lyapunov Fractals

The **Lyapunov exponent** λ is chaos theory's thermometer — it tells you exactly *how chaotic* a system is:

$$\lambda = \lim_{n \to \infty} \frac{1}{n} \sum_{i=0}^{n-1} \ln \left| f'(x_i) \right|$$

| λ value | Meaning |
|:---:|:---|
| **λ < 0** | Trajectories converge → **stable, predictable** |
| **λ = 0** | Neutral stability → **edge of chaos** |
| **λ > 0** | Trajectories diverge exponentially → **CHAOS** |

When we plot λ across a 2D grid of parameters, we get **Lyapunov Fractals** — the order/chaos boundary made visible.

> 🌡️ **THE GOLDEN BOUNDARY**
>
> The contour line where **λ = 0** is the exact frontier between order and chaos.
> On one side: predictability. On the other: infinite sensitivity.
> This boundary is itself a fractal.

```python
# ══════════════════════════════════════════════
#  LYAPUNOV FRACTAL — Zircon City region
# ══════════════════════════════════════════════

def lyapunov_fractal(sequence='AABAB', a_range=(2.0,4.0), b_range=(2.0,4.0),
                     res=800, n_iter=1000, skip=100):
    a_vals = np.linspace(*a_range, res)
    b_vals = np.linspace(*b_range, res)
    A, B = np.meshgrid(a_vals, b_vals)
    x = 0.5 * np.ones((res, res))
    lyap = np.zeros((res, res))
    seq = [0 if c=='A' else 1 for c in sequence]
    for i in range(n_iter):
        r = A if seq[i % len(seq)] == 0 else B
        x = r * x * (1 - x)
        x = np.clip(x, 1e-10, 1-1e-10)
        if i >= skip:
            lyap += np.log(np.abs(r * (1 - 2*x)))
    lyap /= (n_iter - skip)
    return lyap

print("⟳  Computing Lyapunov fractal (sequence=AABAB, 900x900 grid)...", end=" ", flush=True)
lmap = lyapunov_fractal('AABAB', (2.0,4.0), (2.0,4.0), res=900, n_iter=1500, skip=500)
print("✓")
```

## § 08 · Poincaré Sections

### 🔬 Poincaré Sections

A **Poincaré section** slices through a 3D trajectory with a 2D plane — recording every intersection point in the same direction. It's like taking an MRI scan of the attractor.

| Orbit Type | What You See |
|:---|:---|
| **Periodic orbit** | A finite set of dots |
| **Quasi-periodic orbit** | A smooth closed curve |
| **Chaotic orbit** | A **fractal cloud** — the attractor, cross-sectioned |

The **return map** (xₙ → xₙ₊₁) then reveals something stunning — even within the chaos, there is **deterministic structure**. The points don't scatter randomly. They follow a curve.

> 🔬 **HENRI POINCARÉ, 1890**
>
> This technique was invented over 130 years ago — before computers, before chaos theory had a name.
> Poincaré was the first human to *see* deterministic chaos, using only pencil and paper.

```python
# ══════════════════════════════════════════════
#  POINCARE SECTION — Cross-sectioning Lorenz
# ══════════════════════════════════════════════

print("⟳  Solving Lorenz for t=500 (500,000 points)...", end=" ", flush=True)
t_long = np.linspace(0, 500, 500_000)
sol_long = solve_ivp(lorenz, (0, 500), [0.1, 0., 0.],
                     t_eval=t_long, method='DOP853', rtol=1e-9, atol=1e-11)
px, py, pz = sol_long.y
print("✓")

z_level = 27.0
crossings_x, crossings_y = [], []
for i in range(1, len(pz)-1):

        frac = (z_level - pz[i-1]) / (pz[i] - pz[i-1])
        crossings_x.append(px[i-1] + frac*(px[i]-px[i-1]))
        crossings_y.append(py[i-1] + frac*(py[i]-py[i-1]))
crossings_x = np.array(crossings_x)
```
